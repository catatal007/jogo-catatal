Guia de Estudos — O que pode cair

Variáveis

Variáveis são “caixas” que armazenam valores. Em programação usamos variáveis para guardar dados (números, textos, booleanos).

let idade = 17; // idade guarda o valor 17

Condições (IF)

IF permite tomar decisões: se uma condição for verdadeira executa um bloco, caso contrário outro bloco.

if (idade >= 18) { console.log('Maior de idade'); } else { console.log('Menor'); }

Loops

Loops repetem operações: for, while, do...while. Úteis para percorrer vetores ou repetir até condição.

for (let i=0;i<5;i++) { console.log(i); }

Algoritmos

Algoritmo é uma sequência de passos para resolver um problema — pense como receita.

Matemática (ensino médio)

Você verá: equações lineares e quadráticas, progressões (PA/PG), funções, trigonometria básica e análise de gráficos.

HTML & CSS

HTML estrutura o conteúdo (tags como <h1>, <p>, <img>). CSS estiliza (color, margin, display).
